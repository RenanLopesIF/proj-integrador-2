- Ao declarar uma variável
  invés de usar -> `let nome = "usuario1"`;
  use -> `const [nome, setNome] = useState("usuario1")`;

  - Para alterar o valor da variável
    invés de usar ->`nome = "usuario2"`;
    use -> `set("usuario2")`;
